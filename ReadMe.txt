Store Management System - Local Installation

1. Install Python 3.9+ from python.org
2. Double-click 'install_requirements.bat' to install dependencies
3. Double-click 'run_server.bat' to start the application
4. Double-click 'create-superuser.bat' to create an admin user
5. Open your browser to http://localhost:8000 
6. for admin control : http://localhost:8000/admin
 
To stop the server:
- Press Ctrl+C in the command window
- Close the command window
